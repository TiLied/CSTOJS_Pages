Methods

GetPreferredCanvasFormat()

The getPreferredCanvasFormat() method of the
GPU interface returns the optimal canvas texture format for displaying 8-bit depth, standard dynamic range content on the current system.

public GPUTextureFormat GetPreferredCanvasFormat()

Returns

GPUTextureFormat

A string indicating a canvas texture format. The value can be rgba8unorm or bgra8unorm.

Remarks

This is commonly used to provide a Configure(GPUCanvasConfiguration) call with the optimal format value for the current system. This is recommended — if you don&apos;t use the preferred format when configuring the canvas context, you may incur additional overhead, such as additional texture copies, depending on the platform.
