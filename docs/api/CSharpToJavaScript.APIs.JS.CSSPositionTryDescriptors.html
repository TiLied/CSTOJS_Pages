Remarks

Each descriptor in the body of the corresponding @position-try at-rule can be accessed using either its property name in bracket notation or the camel-case version of the property name &quot;propertyName&quot; in dot notation.
For example, you can access the CSS property &quot;property-name&quot; as style[&quot;property-name&quot;] or style.propertyName, where style is a CSSPositionTryDescriptors instance.
A property with a single-word name like height can be accessed using either notation: style[&quot;height&quot;] or style.height.

NOTE

The CSSPositionTryRule interface represents a {{cssxref("@position-try")}} at-rule, and the Style property is an instance of this object.
