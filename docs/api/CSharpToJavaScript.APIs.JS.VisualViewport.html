The VisualViewport interface of the {{domxref(&quot;Visual Viewport API&quot;, &quot;&quot;, &quot;&quot;, &quot;nocode&quot;)}} represents the visual viewport for a given window. For a page containing iframes, each iframe, as well as the containing page, will have a unique window object. Each window on a page will have a unique VisualViewport representing the properties associated with that window.

Remarks

You can get a window's visual viewport using VisualViewport.

NOTE

Note: Only the top-level window has a visual viewport that&apos;s distinct from the layout viewport. Therefore, it&apos;s generally only the VisualViewport object of the top-level window that's useful. For an {{htmlelement("iframe")}}, visual viewport metrics like Width always correspond to layout viewport metrics like ClientWidth.

-Web Viewports Explainer — useful explanation of web viewports concepts, including the difference between visual viewport and layout viewport.
