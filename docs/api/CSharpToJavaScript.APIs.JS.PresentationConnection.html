The PresentationConnection interface of the Presentation API provides methods and properties for managing a single presentation. Each presentation connection is represented by a PresentationConnection object. Both the controlling user agent and receiving user agentMUST implement PresentationConnection.

Properties

BinaryType

When a PresentationConnection object is created, its binaryType IDL attribute MUST be set to the string &quot;arraybuffer&quot;. Upon getting, the attribute MUST return its most recent value (the value it was last set as). Upon setting, the user agent MUST set the IDL attribute to the new value.

public BinaryType BinaryType { get; set; }

Property Value

BinaryType

Remarks

NOTE

Note: The binaryType attribute allows authors to control how binary data is exposed to scripts. By setting the attribute to &quot;blob&quot;, binary data is returned in Blob form; by setting it to &quot;arraybuffer&quot;, it is returned in ArrayBuffer form. The attribute defaults to &quot;arraybuffer&quot;. This attribute has no effect on data sent in a string form.
