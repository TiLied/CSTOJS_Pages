The whenDefined() method of the

public Task<CustomElementConstructor> WhenDefined(string name)

Parameters

Task<CustomElementConstructor>

A Promise that fulfills with the custom element&apos;s constructor when a custom element becomes defined with the given name. If a custom element has already been defined with the name, the promise will immediately fulfill.The promise is rejected with a SyntaxErrorDOMException if the name is not a valid custom element name.
