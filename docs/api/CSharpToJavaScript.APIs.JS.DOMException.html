The DOMException interface represents an abnormal event (called an exception) that occurs as a result of calling a method or accessing a property of a web API. This is how error conditions are described in web APIs.

Remarks

Each exception has a name, which is a short &quot;PascalCase&quot;-style string identifying the error or abnormal condition.

DOMException is a {{Glossary("Serializable object")}}, so it can be cloned with StructuredClone or copied between Workers using PostMessage(dynamic, List<object>).

-A polyfill of DOMException is available in core-js
