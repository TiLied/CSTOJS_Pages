The OscillatorNode interface represents a periodic waveform, such as a sine wave. It is an AudioScheduledSourceNode audio-processing module that causes a specified frequency of a given wave to be created—in effect, a constant tone.

The OscillatorNode() constructor of the Web Audio API creates a new
OscillatorNode object which is an AudioNode that
represents a periodic waveform, like a sine wave, optionally setting the node&apos;s
properties&apos; values to match values in a specified object.
