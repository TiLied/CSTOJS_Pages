Height

The read-only height property of the SVGMaskElement interface returns an SVGAnimatedLength object containing the value of the {{SVGattr("height")}} attribute of the {{SVGElement("marker")}}.

public Unsupported Height { get; }

Property Value

Unsupported

An SVGAnimatedLength object. The baseVal property of this object returns an SVGLength, the value of which returns the height value.

Remarks

NOTE

Note: Although this property is read-only, it is merely a container for two values you can modify, SVGAnimatedLength.BaseVal and SVGAnimatedLength.AnimVal.
