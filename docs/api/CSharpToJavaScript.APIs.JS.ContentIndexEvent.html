Remarks

This event is sent to the global scope of a 'ServiceWorker'. It contains the id of the indexed content to be removed.

The ServiceWorkerGlobalScope.Contentdelete_event event is only fired when the deletion happens due to interaction with the browser's built-in user interface. It is not fired when the 'ContentIndex.Delete' method is called.
