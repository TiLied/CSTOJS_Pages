Remarks

The SVGAngle returned from AnimVal and BaseVal is read only, but the SVGAngle returned from CreateSVGAngle() is writable. When designated as read only, attempts to modify the object will result in an exception being thrown.

An SVGAngle object can be associated with a particular element. The associated element is used to determine which element&apos;s content attribute to update if the object reflects an attribute. Unless otherwise described, an SVGAngle object is not associated with any element.
