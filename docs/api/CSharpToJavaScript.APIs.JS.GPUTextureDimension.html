The dimension read-only property of the
GPUTexture interface represents the dimension of the set of texels for each GPUTexture subresource.

[To("None")]
public enum GPUTextureDimension

Fields

[Value("1d")] _1d = 0
[Value("2d")] _2d = 1
[Value("3d")] _3d = 2

Remarks

This is set via the dimension property in the descriptor object passed into the originating CreateTexture(GPUTextureDescriptor) call, which defaults to &quot;2d&quot; if omitted.
