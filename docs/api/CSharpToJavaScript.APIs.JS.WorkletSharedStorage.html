public Task<Number> RemainingBudget()

Returns

Task<Number>

A Promise that fulfills with a number representing the remaining navigation budget.

Remarks

The navigation budget is the number of entropy bits permitted inside a {{htmlelement("fencedframe")}} due to the SelectURL(string, SharedStorageUrlWithMetadata[], SharedStorageRunOperationMethodOptions) calls per origin every 24 hours. This is not the same as the number of navigations; rather, it is based on the number of potential navigations in each call. Each time a selectURL() navigation occurs, the corresponding origin&apos;s budget decreases by the logarithm (base 2) of the number of URL choices.

Navigation budget is a mechanism designed to limit the rate of leakage of cross-site data to the destination pages navigated to in fenced frames.
