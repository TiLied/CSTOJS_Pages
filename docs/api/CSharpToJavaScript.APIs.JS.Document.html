Remarks

The DOM tree includes elements such as body and table, among many others. It provides functionality globally to the document, like how to obtain the page&apos;s URL and create new elements in the document.

The Document interface describes the common properties and methods for any kind of document. Depending on the document&apos;s type (e.g. HTML, XML, SVG, …), a larger API is available: HTML documents, served with the &quot;text/html&quot; content type, also implement the HTMLDocument interface, whereas XML and SVG documents implement the XMLDocument interface.
