The localStorage read-only property of the Window interface allows you to access a Storage object for the Document's {{glossary("origin")}}; the stored data is saved across browser sessions.

[To("FirstCharToLowerCase")]
public interface WindowLocalStorage

Remarks

localStorage is similar to Window.SessionStorage, except that while localStorage data has no expiration time, sessionStorage data gets cleared when the page session ends — that is, when the page is closed. (localStorage data for a document loaded in a &quot;private browsing&quot; or &quot;incognito&quot; session is cleared when the last &quot;private&quot; tab is closed.)
