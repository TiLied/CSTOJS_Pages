The File and Directory Entries API interface FileSystem is used to represent a file system. These objects can be obtained from the Filesystem property on any file system entry. Some browsers offer additional APIs to create and manage file systems, such as Chrome's Window.RequestFileSystem method.

Remarks

This interface will not grant you access to the users&apos; filesystem. Instead, you will have a &quot;virtual drive&quot; within the browser sandbox if you want to gain access to the users&apos; file system, you need to invoke the user, for example by installing a Chrome extension. The relevant Chrome API can be found here.
