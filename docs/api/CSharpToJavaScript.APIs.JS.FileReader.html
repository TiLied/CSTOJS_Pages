Remarks

File objects may be obtained from a FileList object returned as a result of a user selecting files using the {{HTMLElement("input")}} element, or from a drag and drop operation's DataTransfer object.

FileReader can only access the contents of files that the user has explicitly selected, either using an HTML &lt;input type=&quot;file&quot;&gt; element or by drag and drop. It cannot be used to read a file by pathname from the user&apos;s file system. To read files on the client&apos;s file system by pathname, use the File System Access API. To read server-side files, use standard Ajax solutions, with CORS permission if reading cross-domain.
